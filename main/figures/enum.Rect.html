pub enum Rect<T, Unit> {
    Sized(SizedRect<T, Unit>),
    Extents(ExtentsRect<T, Unit>),
}
Expand description

A 2d rectangle. This type may internally be represented with a SizedRect or an ExtentsRect. All rect types implement Rectlike.

Variants

Sized(SizedRect<T, Unit>)

A SizedRect.

Extents(ExtentsRect<T, Unit>)

An ExtentsRect.
